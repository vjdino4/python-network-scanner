# Python Network Scanner

A multithreaded network scanner built using Python to detect open ports and services on a target system.

## Features
- Scan custom port ranges
- Detect open TCP ports
- Identify common services
- Multithreaded scanning for faster performance
- Save scan results to a file

## Technologies Used
- Python
- Socket Programming
- Multithreading
- Networking Concepts

## How It Works
The scanner attempts to connect to ports on a target host using TCP sockets. If the connection succeeds, the port is marked as open and the associated service is displayed.
